Game Modes

Our developers have years of experience creating and managing successful Minecraft servers, that means we can create almost any game-mode you can think of.

  • For the first version of the server we will create a FFA game mode (Free For All), where all the players drop on a map and fight each other. You will be able to earn $MCR tokens through daily challenges.
  • Our community has decided that the next gamemode will be Survival Games.

Remember that we will add several servers so that you can fight against similar skill players!

We have the ability to add many more game modes, which will be submitted to community vote. Some of them are:

  1. Survival Games
  2. Party games
  3. Tournaments
  4. Among Us
  5. Events
  6. Races
  7. Parkour
  8. Bedwars
  9. Lands and open world
  10. Soccer
  11. Arena 1vs1
  12. Betting 1vs1
  13. Fights against bosses
  14. Clan wars
As you can see, Minecraft does not limit us at all to the creation of new modes, we will always consider the community for decision-making, prioritizing entertainment and the game economy.
